# Assignment-2DSA
# Linear Data Structures Toolkit (LDST)

##  Overview

This project implements fundamental linear data structures from scratch using Python.

The toolkit includes:

* Dynamic Array (with resizing)
* Singly Linked List
* Doubly Linked List
* Stack (using linked list)
* Queue (using linked list)
* Balanced Parentheses Checker

---

##  Features

###  Dynamic Array

* Append elements
* Automatic resizing (capacity doubles when full)
* Pop operation

###  Singly Linked List

* Insert at beginning
* Insert at end
* Delete by value
* Traverse list

###  Doubly Linked List

* Insert after a node
* Delete at position

###  Stack (LIFO)

* Push
* Pop
* Peek

###  Queue (FIFO)

* Enqueue
* Dequeue
* Front

###  Parentheses Checker

* Checks whether an expression is balanced
* Supports (), {}, []

##  Concepts Covered
